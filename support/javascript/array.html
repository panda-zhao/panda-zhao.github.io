数组方法

  • 是不是数组 Array.isArray(arr)
  • 拼接数组 arr.concat(newArr)
  • 数组转换成字符串 arr.join()
  • 查找 元素在数组的下标 arr.indexOf(val)
  • 查找 数组中的指定元素 arr.slice(start, end)
  • 在数组尾部插入新元素 arr.push(val)
  • 在数组头部插入新元素 arr.unshift('a')
  • 删除 数组的第一个元素 arr.shift()
  • 删除 数组的最后一个元素 arr.pop()
  • 排序 arr.sort(function)
  • 替换/添加/删除项目 arr.splice(index, length, newVal)
  • 反转数组中元素的顺序 arr.reverse()
  • 遍历数组
    for...in遍历 index 下标
    for...of遍历 value 值
    forEach遍历 value 和 index
  • 数组去重 Array.from(new Set(arr))
